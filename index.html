 Heading elements are containers for headings and sub-headings. They
have preset text sizes, with h1 being the largest and h6 being the smallest.
Heading elements also help the HTML Document Outline separate the webpage
into sections. The size of the new heading element in relation to the
previous one is what determines the outline as you go down the page.
- If the next heading element is large than the last, a new parent section is
created in the outline.
- If the next heading element is the same size as the last, a new sibling
section is created in the outline.
- If the next heading element is smaller than the last, a new child section is
created in the outline.
